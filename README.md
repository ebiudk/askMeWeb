# ✨ AskMe! Web ✨

## 🚀 プロジェクト概要

AskMe! Web は、VRChatでのあなたの居場所（インスタンス情報）を、信頼できる特定のグループ内でのみ安全に共有するためのウェブアプリケーションです。プライバシーを重視しつつ、フレンドとの合流をスムーズにするための便利なツールとして設計されています。

## 🌟 主な特徴

-   **プライベートな居場所共有**: VRChatのインスタンス情報を、指定したグループメンバーにのみ共有します。
-   **柔軟なグループ管理**: 簡単に新しいグループを作成し、招待コードを使ってメンバーを募集・管理できます。
-   **リアルタイムに近い位置情報更新**: 外部の連携アプリケーションからAPI経由であなたの最新の居場所を自動的に更新します。
-   **セキュアな認証システム**: Auth.js v5 を利用した堅牢な認証で、安心してサービスをご利用いただけます。
-   **パーソナル設定**: ユーザーごとにVRChat IDやAPIキーを管理し、共有設定をカスタマイズできます。

## 📝 サイトの使い方

AskMe! Web を利用して、VRChat での居場所を友達と共有する方法を説明します。

1.  **アカウント作成とログイン**:
    *   初めての方は、連携サービスを利用して簡単にアカウントを作成できます。
    *   すでにアカウントをお持ちの場合は、登録済みの連携サービスでログインしてください。

2.  **グループの作成**:
    *   ログイン後、ダッシュボードから新しいグループを作成できます。
    *   グループ名を設定し、作成すると、一意の「招待コード」が発行されます。

3.  **友達をグループに招待**:
    *   発行された「招待コード」を友達に共有してください。
    *   友達はその招待コードを使ってグループに参加できます。招待URLを直接送ることも可能です。

4.  **VRChatの居場所を共有する準備**:
    *   居場所情報を連携するための「APIキー」を発行します。このAPIキーは、AskMe! Clientなどの外部連携アプリケーションで使用します。

5.  **リアルタイムで居場所を共有**:
    *   AskMe! Client（外部アプリケーション）を起動し、先ほど発行したAPIキーを設定すると、あなたのVRChatでの現在のワールド情報やインスタンス情報が自動的にグループ内で共有されます。
    *   この情報は、グループメンバーのみが見ることができます。

6.  **グループメンバーの居場所を確認**:
    *   ダッシュボードから参加中のグループを選択すると、そのグループに属するメンバーの現在のVRChatでの居場所（ワールド名、最終更新日時など）が一覧で表示されます。
    *   これにより、友達が今どこにいるのかを簡単に確認し、合流することができます。
